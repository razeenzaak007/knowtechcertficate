# **App Name**: CertiMagic

## Core Features:

- Excel Data Upload: Upload an Excel file containing recipient data (Name, WhatsApp Number, etc.).
- Certificate Generation: Generate personalized PDF certificates by merging data from the Excel sheet with a predefined certificate template.
- PDF Upload & Link Creation: Automatically upload generated PDF certificates to a web storage service and create a unique download link for each certificate.
- WhatsApp Link Sending: Send the generated certificate download link to each recipient's WhatsApp number using a WhatsApp Business API integration. As a tool, the LLM should examine each download link to be certain that it works, prior to initiating the message via the WhatsApp Business API
- Template Customization: Allow users to upload and customize certificate templates. Supported format should be PDF and PNG.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a professional and trustworthy feel.
- Background color: Very light lavender (#F0F8FF).
- Accent color: Soft amethyst (#9966CC) for subtle emphasis and calls to action.
- Body font: 'Inter', sans-serif, for clear and readable information.
- Headline font: 'Literata', serif, for a formal and distinguished title.
- Use clean and professional icons for key actions and data visualizations.
- Maintain a clean and well-structured layout for easy navigation.
